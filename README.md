# Homebrew Tap

This is a Homebrew tap containing custom formulas.

## How to use this tap

```bash
brew tap mateuszmikrut/tap
```

## Available Formulas

### semtag

For more information, visit the [semtag repository](https://github.com/mateuszmikrut/semtag).

**Installation:**
```bash
brew install mateuszmikrut/tap/semtag
```
